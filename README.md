## 💡 Project: SkillPilot – Agentic AI for Personalized Course Pathways

### 🔍 Problem Statement

With an overwhelming number of online resources, students often struggle to find the right learning path. The lack of guidance and dynamic support results in poor learning outcomes.

### 💡 Solution Overview

**SkillPilot** is an intelligent Agentic AI system that guides students by:
- Identifying user interests and current skill levels  
- Recommending tailored course roadmaps  
- Dynamically adapting based on progress  



## ⚙️ Technology Stack

- **Cloud Platform:** IBM Watsonx, IBM Cloud Lite  
- **Foundation Model:** Granite LLMs (e.g., LLaMA-3-70b-instruct)  
- **Frameworks & Tools:**  
  - LangGraph (ReAct architecture)  
  - Prompt Hub (Dynamic Prompting)  
  - IBM Agent Lab (Deployment)  



## 🚀 Project Workflow

### 1. **Interest & Skill Assessment**  
- Gathers user input via chat  
- Detects interest areas like Frontend, Cybersecurity, etc.  
- Estimates current skill level using natural language understanding  

### 2. **Roadmap Generation**  
- Suggests a roadmap from uploaded or predefined course materials  
- Uses LangGraph agents and Watsonx conversation models  
- Updates based on progress & user feedback  

### 3. **Deployment**  
- Deployed using IBM Watsonx Agent Lab  
- Interactive conversation interface  
- Provides real-time learning path suggestions  


## 📊 Results

SkillPilot successfully:
- Provided dynamic course suggestions based on individual interests  
- Demonstrated agent memory for better personalization  
- Validated real-time interaction and adaptability through IBM Watsonx
